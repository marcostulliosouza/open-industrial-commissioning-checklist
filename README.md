# Open Industrial Commissioning Checklist (OICC)

OICC is an open-source tool designed to organize, track and document
industrial commissioning activities in a structured and auditable way.

## Features
- Industrial commissioning checklists
- Evidence attachment (photos, logs)
- Tag-based validation (I/O, robot signals, process tags)
- Status tracking (OK / NOK / Pending)
- Final commissioning report (PDF)

## Target Users
- Automation engineers
- System integrators
- OEMs
- Industrial training centers
